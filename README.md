# Water Love Tracker

A romantic, Valentine's Day-themed water tracking application designed as a special gift. 

## Features
- **Daily Goal Tracking**: Helps track water intake with a visual progress bar.
- **Unlockable Love Letters**: Completing daily goals unlocks personal love letters, leading up to a special Valentine's Day message.
- **Romantic Theme**: Beautiful pastel aesthetics, floating hearts, and custom animations.
- **Responsive Design**: Works perfectly on both desktop and mobile devices.

Built with simple HTML, CSS, and Vanilla JavaScript.
